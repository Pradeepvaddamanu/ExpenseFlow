# ExpenseFlow

ExpenseFlow is a Smart Group Expense Management System that helps users manage shared expenses, calculate optimized settlements, and generate AI-powered spending insights.

## Features

- User Authentication (JWT)
- Group Expense Management
- Expense Tracking
- Smart Settlement Calculation
- AI Expense Insights
- Cloud Database Integration

## Tech Stack

### Frontend
- React.js
- Vite
- Tailwind CSS
- Axios

### Backend
- FastAPI
- Python
- SQLAlchemy
- JWT Authentication

### Database
- PostgreSQL (Neon)

### AI
- Groq API
- Llama 3.3 70B
